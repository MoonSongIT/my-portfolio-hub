## PowerShell에서 Claude API 프록시 테스트 절차

### 1단계: 서버 시작

PowerShell 터미널 **첫 번째 창**을 열고:

```powershell
cd C:\Users\dohay\ClaudeWork\my-portfolio-hub
npm run server
```

`🤖 Claude 프록시 서버: http://localhost:3001` 메시지가 나오면 성공!

> ⚠️ 이 창은 닫지 마세요 (서버가 계속 실행되어야 합니다)

---

### 2단계: API 호출 테스트

PowerShell 터미널 **두 번째 창**을 열고:

```powershell
Invoke-RestMethod -Uri "http://localhost:3001/api/claude" -Method POST -ContentType "application/json" -Body '{"systemPrompt":"당신은 도움이 되는 AI입니다. 한 문장으로 짧게 답하세요.","messages":[{"role":"user","content":"안녕하세요"}]}'
```

---

### 3단계: 응답 확인

**성공하면** 이런 형태의 응답이 나옵니다:

```
model        : claude-sonnet-4-6
id           : msg_01Xxxx...
type         : message
role         : assistant
content      : {@{type=text; text=안녕하세요! ...}}
stop_reason  : end_turn
```

**`content` 안의 텍스트만 보고 싶으면:**

```powershell
$response = Invoke-RestMethod -Uri "http://localhost:3001/api/claude" -Method POST -ContentType "application/json" -Body '{"systemPrompt":"당신은 도움이 되는 AI입니다.","messages":[{"role":"user","content":"안녕하세요"}]}'

$response.content[0].text
```

---

### 4단계: 에러 케이스 테스트 (선택)

**빈 메시지 전송 (에러 처리 확인):**

```powershell
Invoke-RestMethod -Uri "http://localhost:3001/api/claude" -Method POST -ContentType "application/json" -Body '{"systemPrompt":"테스트","messages":[]}'
```

**헬스체크:**

```powershell
Invoke-RestMethod -Uri "http://localhost:3001/api/health"
```

`status: ok` 가 나오면 서버 정상!

---

### 5단계: 서버 종료

첫 번째 터미널에서 `Ctrl + C` 를 누르면 서버가 종료됩니다.
