## 1. 프로젝트 개요

```
프로젝트명  : My Portfolio Hub
목적        : 매매 심리를 기록하고 AI로 돌아보는 개인 투자 성장 도구
대상 사용자 : 단순 수익 추적을 넘어 스스로의 투자 스타일을 개선하고 싶은 개인 투자자
GitHub      : https://github.com/MoonSongIT/my-portfolio-hub
```

### 이 앱이 해결하는 문제

증권사 앱은 숫자(수익/손실)는 보여주지만, 왜 그 결정을 했는지는 기록할 수 없다.
같은 실수(추격매매, 공포 매도 등)가 반복되어도 패턴을 인식하기 어렵고, 잘된 투자 결정도 기억에서 사라진다.

> 핵심 가치: 수익 관리 도구가 아닌, 투자 습관 성장 도구

### 핵심 기능 (우선순위 순)

| # | 기능 | 설명 | 대상 Phase |
|---|------|------|-----------|
| 1 | **투자 매매 일지** | 매매 직후/마감 후 종목별 심리 기록 (선택형 카테고리) | Phase 2 ← 핵심 |
| 2 | 포트폴리오 현황 | 보유 종목 입력, 매수가 기록, 실시간 수익률 계산 | Phase 2~3 |
| 3 | **AI 코치 채팅** | 내 일지 데이터 기반 매매 패턴 분석·피드백 | Phase 4 ← 차별점 |
| 4 | 성과 추적 | 심리 유형별 수익률 비교, 벤치마크 비교 | Phase 2~5 |
| 5 | 종목 탐색 / 관심종목 | 신규 투자 후보 검색, 뉴스·공시 모니터링 | Phase 3~5 |

### 매매 심리 카테고리 (선택형)

```
매수 심리: 미래가치 투자 | 분할매수 원칙 | 추격매매 | 뉴스 편승 | 저가 매수 | 목표가 도달 | 기타
매도 심리: 목표가 실현 | 손절 원칙 | 공포에 매도 | 수익 실현 (조급) | 리밸런싱 | 기타
```

### 최종 기술 스택

```
Frontend   : React 18 + Vite 6 (SWC)
스타일     : Tailwind CSS + shadcn/ui (Radix UI)
차트       : Recharts (기본) + lightweight-charts v4 (캔들스틱)
상태관리   : Zustand v5 + Immer (클라이언트) / TanStack Query v5 (서버)
라우팅     : React Router v6
주가 API   : Yahoo Finance (비공식) / KRX / 한국투자증권 OpenAPI
AI         : Anthropic Claude API (claude-sonnet-4-6) — 서버사이드 프록시
저장소     : LocalStorage → IndexedDB (Dexie.js) → Supabase
배포       : Vercel (프론트) + Vercel Serverless Functions (프록시)
```

### 5.1 Step 1: Vercel Serverless 프록시 구축

```
프론트엔드 → /api/claude → Vercel Serverless → Anthropic API
                              (API 키 보호)
```

```javascript
// api/claude.js (Vercel Serverless Function)
export default async function handler(req, res) {
  if (req.method !== 'POST') return res.status(405).end();

  const response = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': process.env.ANTHROPIC_API_KEY,
      'anthropic-version': '2023-06-01',
    },
    body: JSON.stringify({
      model: 'claude-sonnet-4-6',
      max_tokens: 4096,
      system: req.body.systemPrompt,
      messages: req.body.messages,
    }),
  });

  const data = await response.json();
  res.json(data);
}
```

export async function sendToAgent(agentType, userMessage, context = {}) {
  const systemPrompt = getAgentPrompt(agentType, context);
  const response = await axios.post('/api/claude', {
