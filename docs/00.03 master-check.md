- ⬜ Vite 6 + React 18 프로젝트 생성 (`npm create vite@latest`)
- ⬜ `@vitejs/plugin-react-swc` 설치 및 설정
- ⬜ `@/` 경로 별칭 (alias) 설정 (`vite.config.js` + `jsconfig.json`)
- ⬜ `npm run dev` 정상 실행 확인

### 1.2 Tailwind CSS + shadcn/ui

- ⬜ Tailwind CSS 설치 및 설정
- ⬜ `index.css`에 Tailwind 디렉티브 추가
- ⬜ shadcn/ui 초기화 (`npx shadcn@latest init`)
- ⬜ 기본 컴포넌트 설치: Button, Card, Table, Dialog, Tabs, Select, Command, Tooltip, Sheet, Input, Label, Badge, Separator, DropdownMenu

- ⬜ `zustand` + `immer` 설치
- ⬜ `@tanstack/react-query` + devtools 설치
- ⬜ `react-router-dom` 설치
- ⬜ `recharts` 설치
- ⬜ `axios` 설치
- ⬜ `lucide-react` 설치
