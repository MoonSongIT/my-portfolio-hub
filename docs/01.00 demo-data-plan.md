## 3. 거래 규칙

### 3-1. 거래일

- **기간**: 2026-02-16 ~ 2026-04-15 (약 2개월)
- **거래일**: 월~금 (주말 제외)
- **법정공휴일 제외**: 2026-03-02 (삼일절 대체공휴일)
- **예상 거래일수**: ~42일

### 3-2. 거래 빈도

- 1일 평균 2건 (매수 + 매도 혼합)
- 양 계좌 합산 기준

### 3-3. 거래 페이즈

| 페이즈 | 기간 | 전략 |
|--------|------|------|
| Phase 1 (초기 매수) | Day 1~6 | 초기 자본의 80% 투자, 종목별 분할매수 |
| Phase 2 (매매) | Day 7~38 | 매수/매도 혼합, 수익 실현 + 저가 재매수 |
| Phase 3 (보유) | Day 39~42 | 거래 없음, 포지션 유지 |

### 3-4. 수익률 목표

- **전체 순이익**: 초기 자본 대비 ~2% 증가
- **일반계좌**: ~400,000원 수익 (실현 + 미실현)
- **ISA계좌**: ~400,000원 수익 (실현 + 미실현)
- **총 목표**: ~800,000원 / 40,000,000원 = 2%

### 3-5. 주가 시뮬레이션

- 결정론적(deterministic) 가격 모델 사용 (재현 가능)
- 선형 상승 트렌드 + 사인파 주기적 변동
- 종목별 고유 위상(phase) 적용으로 가격 패턴 차별화

```
price = basePrice × (1 + totalReturn × progress) + basePrice × volatility × sin(day × 0.3 + phase)
```

### 3-6. 매매 심리 카테고리 분배

**매수 심리:**
- 미래가치 투자, 분할매수 원칙 (초기 매수 시)
- 저가 매수 (하락 시 추가매수)
- 추격매매, 뉴스 편승 (일부 비합리적 매수)

**매도 심리:**
- 목표가 실현 (수익 매도 시)
- 수익 실현 (조급) (조기 매도 시)
- 손절 원칙, 공포에 매도 (손실 매도 시)
- 리밸런싱

## 4. 데이터 흐름

```
1. Login.jsx → "데모 로그인" 버튼 클릭
2. authStore → 데모 사용자 등록/로그인
3. seedDemoData() 호출
   ├─ accountStore.addAccount() × 2 (일반/ISA)
   ├─ cashFlowStore.addCashFlow() × 2 (초기입금)
   └─ journalStore.addEntry() × ~80 (거래)
       ├─ 매도 시 pnl 자동 계산
       └─ cashFlowStore.addAutoFlow() 자동 생성
4. IndexedDB에 비동기 저장
5. 홈(/)으로 이동
```

## 6. 주의사항

- 데모 데이터는 **최초 1회만** 생성 (중복 방지: 계좌 ID 존재 여부 확인)

- NVDA(USD) 거래는 소량으로 제한 (환전 로직 미구현)
