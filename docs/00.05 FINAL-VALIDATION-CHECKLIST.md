## T1️⃣ E2E 흐름 테스트: 입금 → 매수 → 매도 → 출금

### 목표
입금부터 출금까지 전체 거래 흐름에서 모든 상태 업데이트가 정확하게 이루어지는지 검증

### 테스트 환경
- ✅ 앱 실행: `npm run dev`
- ✅ 브라우저 열기: `http://localhost:5173`
- ✅ DevTools 열기: **F12** → **Console** 탭 활성화
- ✅ 로그인 (샘플 사용자 계정 사용)

### 테스트 시나리오

#### Step 1: 초기 상태 확인
1. 대시보드 접속
2. **개발자 도구 → Console** 에서 다음 명령어 실행:
   ```javascript
   // Zustand 상태 확인 (모든 store 조회)
   console.log("포트폴리오 스토어:", JSON.parse(localStorage.getItem('portfolio-storage')));
   console.log("현금 흐름 스토어:", JSON.parse(localStorage.getItem('cashflow-storage')));
   console.log("거래 일지 스토어:", JSON.parse(localStorage.getItem('journal-storage')));
   ```
3. **예상 결과**:
   - `portfolio-storage`: `selectedAccountId: 'all'`, `exchangeRate: 1350`
   - `cashflow-storage`: `cashFlows: []` (초기 데이터 없음)
   - `journal-storage`: `entries: []` 또는 샘플 데이터

#### Step 2: 입금 거래 추가
1. **좌측 메뉴** → **"현금 흐름"** (또는 대시보드 우측 상단 카드 클릭)
2. **"입금 추가"** 버튼 클릭
3. 다음 정보 입력:
   - 날짜: `2026-04-13`
   - 계좌: 기본 계좌 선택 (예: "My Account")
   - 유형: **"입금"**
   - 금액: **5,000,000 원** (500만 원)
   - 메모: "초기 자본금"
4. **"저장"** 클릭
5. **확인**:
   ```javascript
   // 콘솔에서 실행
   const cf = JSON.parse(localStorage.getItem('cashflow-storage'));
   console.log("입금 후 현금흐름:", cf.state.cashFlows);
   // 예상: type: 'deposit', amount: 5000000, isAuto: false

   const avail = cf.state.cashFlows.filter(f => f.type === 'deposit').reduce((s, f) => s + f.amount, 0);
   console.log("총 입금액:", avail); // 5,000,000
   ```

#### Step 3: 매수 거래 추가 (투자 가능 금액에서 차감)
1. **좌측 메뉴** → **"매매 일지"**
2. **"거래 추가"** (또는 **"빠른 입력"**) 클릭
3. 다음 정보 입력:
   - 날짜: `2026-04-13`
   - 계좌: 위에서 선택한 계좌와 동일
   - 티커: `005930` (삼성전자)
   - 종목명: `삼성전자`
   - 시장: `KRX`
   - 매매: **"매수"**
   - 현재가: `72,000` 원
   - 수량: `10` 주
   - 수수료: `5,000` 원 (선택사항)
   - 심리 카테고리: **"미래가치 투자"**
   - 메모: "기술주 핵심 보유"
4. **"저장"** 클릭
5. **확인**:
   ```javascript
   // 거래 일지 확인
   const journal = JSON.parse(localStorage.getItem('journal-storage'));
   console.log("거래 항목:", journal.state.entries);
   // 예상: action: 'buy', quantity: 10, price: 72000, fee: 5000

   // 자동 생성된 현금흐름 확인 (매수 자동 연동)
   const cf = JSON.parse(localStorage.getItem('cashflow-storage'));
   const autoFlows = cf.state.cashFlows.filter(f => f.isAuto === true);
   console.log("자동 현금흐름:", autoFlows);
   // 예상: type: 'withdrawal', amount: 720000 + 5000 = 725000, isAuto: true

   // 투자 가능 금액 재계산
   const totalDeposit = cf.state.cashFlows
     .filter(f => f.type === 'deposit' && !f.isAuto)
     .reduce((s, f) => s + f.amount, 0);
   const totalWithdrawal = cf.state.cashFlows
     .filter(f => f.type === 'withdrawal' && !f.isAuto)
     .reduce((s, f) => s + f.amount, 0);
   console.log("투자 가능 금액:", totalDeposit - totalWithdrawal - 725000);
   // 예상: 5,000,000 - 0 - 725,000 = 4,275,000
   ```

#### Step 4: 대시보드 KPI 카드 검증
1. **좌측 메뉴** → **"대시보드"**
2. 다음 정보 확인:
   - **"총 자산"** 카드: `5,000,000 원` (현금 + 보유 종목 평가금액)
   - **"보유 종목"** 카드: `1종목` (삼성전자)
   - **"투자 가능 금액"** 카드 (맨 오른쪽): `4,275,000 원` ✅
3. **세부 확인**:
   ```javascript
   // 현재 보유 종목 확인
   const portfolio = JSON.parse(localStorage.getItem('portfolio-storage'));
   const journal = JSON.parse(localStorage.getItem('journal-storage'));

   // portfolio.state.prices에 실시간 시세 있는지 확인

   // 보유 종목 목록

   ```

#### Step 5: 매도 거래 추가 (수익/손실 계산)
1. **매매 일지** → **"거래 추가"**
2. 다음 정보 입력:
   - 날짜: `2026-04-13` (같은 날)
   - 계좌: 동일 계좌
   - 티커: `005930`
   - 종목명: `삼성전자`
   - 매매: **"매도"**
   - 현재가: `73,000` 원 (1,000원 상승)
   - 수량: `5` 주 (절반만 매도)
   - 심리 카테고리: **"목표가 실현"**
   - 메모: "부분 익절"
3. **"저장"** 클릭
4. **확인**:
   ```javascript
   // 매도 거래 시 자동 손익 계산 확인
   const journal = JSON.parse(localStorage.getItem('journal-storage'));
   const sellEntry = journal.state.entries.find(e => e.action === 'sell');
   console.log("매도 거래:", sellEntry);
   // 예상: pnl: (73000 - 72000) * 5 = 5,000원

   // 매도 자동 현금흐름 (deposit)
   const cf = JSON.parse(localStorage.getItem('cashflow-storage'));
   const sellFlow = cf.state.cashFlows.find(f => f.isAuto && f.type === 'deposit');
   console.log("매도 입금:", sellFlow);
   // 예상: amount: 73000 * 5 = 365,000 (수수료 제외)

   // 보유 수량 변경 확인 (10주 → 5주)
   const holdings = journal.state.entries.reduce((acc, e) => {
     const key = e.ticker;
     if (!acc[key]) acc[key] = 0;
     acc[key] += e.action === 'buy' ? e.quantity : -e.quantity;
     return acc;
   }, {});
   console.log("남은 보유 수량 (삼성전자):", holdings['005930']); // 5주
   ```

#### Step 6: 출금 거래 추가 (현금화)
1. **현금 흐름** → **"출금 추가"**
2. 다음 정보 입력:
   - 날짜: `2026-04-13`
   - 계좌: 동일 계좌
   - 유형: **"출금"**
   - 금액: **1,000,000 원** (부분 출금)
   - 메모: "생활비 출금"
3. **"저장"** 클릭
4. **최종 확인**:
   ```javascript
   const cf = JSON.parse(localStorage.getItem('cashflow-storage'));

   // 최종 현금 흐름 요약
   const deposits = cf.state.cashFlows
     .filter(f => f.type === 'deposit' && !f.isAuto)
     .reduce((s, f) => s + f.amount, 0);
   const withdrawals = cf.state.cashFlows
     .filter(f => f.type === 'withdrawal' && !f.isAuto)
     .reduce((s, f) => s + f.amount, 0);
   const autoWithdrawals = cf.state.cashFlows
     .filter(f => f.type === 'withdrawal' && f.isAuto)
     .reduce((s, f) => s + f.amount, 0);
   const autoDeposits = cf.state.cashFlows
     .filter(f => f.type === 'deposit' && f.isAuto)
     .reduce((s, f) => s + f.amount, 0);

   console.log("=== 최종 현금 흐름 정산 ===");
   console.log("수동 입금:", deposits); // 5,000,000
   console.log("수동 출금:", withdrawals); // 1,000,000
   console.log("자동 입금 (매도):", autoDeposits); // 365,000
   console.log("자동 출금 (매수):", autoWithdrawals); // 725,000
   console.log("순 현금 흐름:", deposits - withdrawals + autoDeposits - autoWithdrawals);
   // 예상: 5,000,000 - 1,000,000 + 365,000 - 725,000 = 3,640,000
   ```

## T2️⃣ 다중 계좌 투자 가능 금액 계산

### 목표
여러 계좌를 전환할 때 각 계좌의 투자 가능 금액이 독립적으로 정확하게 계산되는지 검증

### 테스트 환경
- ✅ 앱이 실행 중인 상태
- ✅ T1 테스트 완료 후 진행 (데이터가 있는 상태)

### 테스트 시나리오

#### Step 1: 새 계좌 추가
1. **좌측 메뉴** → **"계좌 관리"** (또는 설정 메뉴)
2. **"계좌 추가"** 버튼 클릭
3. 다음 정보 입력:
   - 계좌명: `My Account 2` (또는 다른 이름)
   - 증권사: `한국투자증권` (선택)
   - 계좌 유형: **"ISA"**
   - 통화: **"KRW"**
4. **"저장"** 클릭


8. **확인**:
   ```javascript
   // 계좌2의 거래만 필터링
   const journal = JSON.parse(localStorage.getItem('journal-storage'));
   const acct2Id = JSON.parse(localStorage.getItem('account-storage'))
     .state.accounts.find(a => a.name === 'My Account 2')?.id;

   const acct2Entries = journal.state.entries.filter(e => e.accountId === acct2Id);
   console.log("계좌2 거래:", acct2Entries);
   // 예상: 1개 매수 거래 (SK하이닉스)

   // 계좌2의 투자 가능 금액 계산
   const cf = JSON.parse(localStorage.getItem('cashflow-storage'));
   const acct2Deposits = cf.state.cashFlows
     .filter(f => f.accountId === acct2Id && f.type === 'deposit' && !f.isAuto)
     .reduce((s, f) => s + f.amount, 0);
   const acct2Withdrawals = cf.state.cashFlows
     .filter(f => f.accountId === acct2Id && f.type === 'withdrawal' && !f.isAuto)
     .reduce((s, f) => s + f.amount, 0);
   const acct2Cost = acct2Entries
     .filter(e => e.action === 'buy')
     .reduce((s, e) => s + e.price * e.quantity + (e.fee || 0), 0);

   const acct2Available = acct2Deposits - acct2Withdrawals - acct2Cost;
   console.log("계좌2 투자 가능 금액:", acct2Available);
   // 예상: 3,000,000 - 0 - (60,000*20 + 3,000) = 3,000,000 - 1,203,000 = 1,797,000
   ```

   ```javascript
   // 현재 보유 현황

     .filter(e => e.accountId === acct1Id)
     .reduce((acc, e) => {
       if (!acc[e.ticker]) {
         acc[e.ticker] = {
           qty: 0,
           cost: 0,
           name: e.name,
           market: e.market
         };
       }

         acc[e.ticker].qty -= e.quantity;
       }
       return acc;
     }, {});

   // 예상:
   ```
