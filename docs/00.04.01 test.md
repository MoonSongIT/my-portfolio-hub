# My Portfolio Hub — 통합 테스트 시나리오

> **목적**: 현재까지 구현된 모든 기능을 하나의 흐름으로 검증한다.
> **환경**: `npm run dev` 실행 후 `http://localhost:5173` 접속
> **소요시간**: 약 30~40분
> **작성일**: 2026-04-08

---

## STEP 1 — 회원가입 & 로그인

### 1-1. 신규 회원가입

1. 브라우저에서 `http://localhost:5173` 접속
2. 로그인 화면이 표시되는지 확인 (비로그인 상태에서 `/` 접속 시 `/login` 리다이렉트)
3. **"계정이 없으신가요? 회원가입"** 링크 클릭
4. 아래 정보 입력:
   - 이름: `홍길동`
   - 이메일: `test@portfolio.com`
   - 비밀번호: `test1234`
5. **회원가입** 버튼 클릭

**기대 결과**

- [X] 오류 없이 가입 완료
- [X] 자동으로 대시보드(`/`) 로 이동
- [X] 상단 헤더에 "홍길동" 이름 표시

### 1-2. 로그아웃 & 재로그인

1. 헤더 우측 **로그아웃(→)** 아이콘 클릭
2. `/login` 으로 이동되는지 확인
3. 이메일 `test@portfolio.com`, 비밀번호 `test1234` 입력
4. **로그인** 클릭

**기대 결과**

- [X] 대시보드로 이동
- [X] 잘못된 비밀번호 입력 시 오류 메시지 표시

## STEP 2 — 계좌 추가

### 2-1. 첫 번째 계좌 추가 (계좌 없는 상태에서 진입)

1. 좌측 사이드바 **포트폴리오** 메뉴 클릭
2. 계좌 선택기 위치에 `+ 계좌를 먼저 추가해주세요` 버튼이 보이는지 확인
3. 해당 버튼 클릭 → **계좌 관리** 모달 오픈 확인
4. 모달에서 **계좌 추가** 버튼 클릭
5. 아래 정보 입력:
   - 계좌 이름: `키움 주식계좌`
   - 증권사: `키움증권`
   - 계좌 유형: `일반(위탁)`
   - 기본 통화: `원화 (KRW)`
6. **추가** 버튼 클릭 → 목록으로 복귀
7. **닫기**로 모달 닫기

**기대 결과**

- [X] 계좌 목록에 "키움 주식계좌" 항목 표시
- [X] 포트폴리오 페이지 계좌 선택기에 "키움 주식계좌 · 일반(위탁)" 버튼 표시

### 2-2. 두 번째 계좌 추가 (계좌가 있는 상태)

1. 포트폴리오 페이지 계좌 선택기 우측 **⚙️(톱니바퀴)** 아이콘 클릭
2. **계좌 추가** 버튼 클릭
3. 아래 정보 입력:
   - 계좌 이름: `미래에셋 ISA`
   - 증권사: `미래에셋증권`
   - 계좌 유형: `ISA`
   - 기본 통화: `원화 (KRW)`
4. **추가** 후 **닫기**

**기대 결과**

- [X] 계좌 선택기에 `전체 (2계좌)`, `키움 주식계좌`, `미래에셋 ISA` 버튼 3개 표시
- [X] 각 계좌 버튼에 유형 배지(일반(위탁) / ISA) 색상 구분 표시

### 2-3. 계좌 수정 확인

1. ⚙️ 클릭 → 목록에서 `키움 주식계좌` 우측 ✏️ 클릭
2. 메모 필드에 `주 거래 계좌` 입력
3. **저장** 클릭

**기대 결과**

- [X] 저장 후 목록에서 변경 내용 반영 확인

## STEP 3 — 자금 관리 (입출금)

### 3-1. 키움 계좌에 초기 자금 입금

1. 사이드바 **자금 관리** 메뉴 클릭
2. 계좌 선택 드롭다운에서 `키움 주식계좌 · 일반(위탁)` 선택
3. 선택 즉시 계좌 정보 카드가 우측에 표시되는지 확인 (계좌명, 증권사, 통화, 메모)
4. 상단 **입금** 버튼 클릭
5. 아래 정보 입력:
   - 계좌: `키움 주식계좌`
   - 금액: `10,000,000`
   - 날짜: 오늘
   - 메모: `초기 투자금`
6. **저장** 클릭

**기대 결과**

- [X] 입출금 내역에 `+₩10,000,000` 행 추가
- [X] 자금 현황 카드에 **총 입금액 ₩10,000,000**, **투자 가능 금액 ₩10,000,000** 표시

### 3-2. 미래에셋 ISA 계좌 입금

1. 계좌 드롭다운에서 `미래에셋 ISA` 선택
2. 입금 버튼 클릭 → 금액 `5,000,000`, 메모 `ISA 납입` 입력 후 저장

**기대 결과**

- [X] ISA 계좌 투자 가능 금액 `₩5,000,000` 표시

### 3-3. 전체 계좌 합산 확인

1. 드롭다운에서 **전체** 선택
2. 자금 현황 카드에 합산 금액 확인

**기대 결과**

- [X] 총 입금액 `₩15,000,000` 표시 (두 계좌 합산)
- [X] 입출금 내역에 두 계좌 내역 모두 표시

- [ ] 자산 배분 파이차트 업데이트 확인
