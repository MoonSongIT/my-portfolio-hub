#### Step 4: Dashboard 오늘 손익 카드 확인
UI 경로: Dashboard → "오늘 손익" 카드 확인

시각적 확인:
```
오늘 손익: [금액] [수익률%]
  색상: 양수면 green, 음수면 red
  아이콘: 양수면 TrendingUp, 음수면 TrendingDown
```

콘솔 검증:
```javascript
// 오늘 손익 계산 (Dashboard.jsx의 dailyChange 메모이즈 훅)
const dailyChange = (() => {
  const holdings = portfolioStore.getState().getSelectedHoldings();
  const snapshots = dailyPnlStore.getState().getSnapshotsByDate(today, 'acc-1');

  if (snapshots.length > 0) {
    const snapshotPnl = snapshots.reduce((sum, s) => sum + (s.dailyPnl || 0), 0);
    const totalValue = holdings.reduce((sum, h) => sum + h.currentPrice * h.quantity, 0)
                     + portfolioStore.getState().getSelectedCash().krw;
    const prevTotal = totalValue - snapshotPnl;
    return { amount: snapshotPnl, rate: (snapshotPnl / prevTotal) };
  }
  return { amount: 0, rate: 0 };
})();

console.log('Daily Change:', dailyChange);
// → {amount: [손익액], rate: [수익률]}
```

**예상 결과**:
- 오늘 손익 표시 ✓
- 스냅샷 기반 계산 (또는 previousClose 폴백) ✓
- 색상/아이콘 조건부 표시 ✓

4. 투자 가능 금액 → getSelectedCash() (입금 - 출금 - 투자원가)
