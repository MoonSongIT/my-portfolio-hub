## 4. Sprint 1 — 차트 고도화

> **목표**: 금융 전문가 수준의 기술적 분석 차트
> **예상 기간**: 1~2일
> **브랜치**: `feature/phase5-chart-advanced`

### 4.1 추가 기술 지표

**Bollinger Bands (볼린저 밴드)**
```
구현 위치: src/components/charts/CandlestickChart.jsx
계산 로직: src/utils/indicators.js (신규)

- 중심선: SMA(20)
- 상단밴드: SMA(20) + 2σ
- 하단밴드: SMA(20) - 2σ
- 3개 LineSeries 추가 (상단: 회색 점선, 중심: 파랑, 하단: 회색 점선)
- AreaSeries로 밴드 영역 표시 (반투명)
```

**RSI (Relative Strength Index)**
```
구현: 별도 패널 (차트 하단 분리 영역)
기간: 14일 기본
과매수/과매도 기준선: 70/30

- 별도 createChart() 인스턴스 (높이 100px)
- 메인 차트와 시간축 동기화 (subscribeVisibleTimeRangeChange)
- LineSeries (보라색)
- 기준선 2개 (70: 빨강 점선, 30: 초록 점선)
```

**MACD (Moving Average Convergence Divergence)**
```
구현: 별도 패널 (RSI 아래 또는 토글 전환)
- MACD 라인: EMA(12) - EMA(26)
- 시그널 라인: EMA(9) of MACD
- 히스토그램: MACD - Signal (HistogramSeries)

색상:
- MACD 라인: 파랑
- 시그널: 주황
- 히스토그램 양수: 초록, 음수: 빨강
```

### 4.2 다중 타임프레임

```
현재: 일봉(1D)만 지원
추가: 주봉(1W), 월봉(1M)

구현 방식:
- 타임프레임 선택 탭 UI (1D | 1W | 1M)
- 일봉 데이터를 주봉/월봉으로 리샘플링 (utils/indicators.js)
  - 주봉: 월~금 OHLCV 합산
  - 월봉: 해당 월 OHLCV 합산
- 선택 시 차트 데이터 교체 (setData)
```

### 4.3 매매 일지 연동 마커

```
핵심 차별점: 종목 차트 위에 내 매매 기록을 시각화

구현:
- journalStore에서 해당 ticker의 entries 조회
- 매수 마커: 🔺 초록색 arrowUp (하단)
- 매도 마커: 🔻 빨간색 arrowDown (상단)
- 마커 tooltip: "매수 72,000원 × 10주 | 추격매매"
- series.setMarkers([...]) API 활용

데이터 흐름:
journalStore.entries
  → filter(ticker === currentTicker)
  → map to { time, position, color, shape, text }
  → series.setMarkers(markers)
```

### 4.4 차트 기간 선택

```
현재: 전체 기간만 표시
추가: 1개월 | 3개월 | 6개월 | 1년 | 전체

구현:
- 버튼 그룹 UI (shadcn/ui Tabs 또는 ButtonGroup)

```

### 4.5 지표 토글 UI

```
구현: 지표 선택 패널 (체크박스 그룹)

☑ MA 20  ☑ MA 60  ☑ MA 120
☐ Bollinger Bands
☐ RSI
☐ MACD
☐ 거래량

- 각 지표의 show/hide 상태 관리 (useState 또는 settingsStore)
- 토글 시 해당 시리즈 applyOptions({ visible: true/false })
- RSI/MACD는 패널 자체 mount/unmount
```

## 6. Sprint 3 — Watchlist 페이지 완성

> **목표**: 실시간 알림 조건 + AlertAgent 시장 브리핑 + 브라우저 알림
> **예상 기간**: 1~2일
> **브랜치**: `feature/phase5-watchlist`

### 6.1 가격 알림 조건 설정

```
현재: AlarmDialog 컴포넌트 존재 (기본 UI)
고도화:

watchlistStore 확장:
{
  items: [
    {
      ticker: "005930",
      name: "삼성전자",
      alerts: [
        { id: "a1", type: "target_price", direction: "above", value: 80000, enabled: true },
        { id: "a2", type: "percent_change", direction: "below", value: -5, enabled: true },
        { id: "a3", type: "52week_high", enabled: false }
      ]
    }
  ]
}

알림 유형:
```
