> **기간**: 5~7일

## Phase 2 아키텍처 개요

### 데이터 모델: 사용자 → 계좌 → 종목

```

│  User (사용자)                                                │
│  ├── id, name, email, password(해시)                          │
│  ├── Account[] (계좌 목록)                                    │
│  │   ├── 일반(위탁) 계좌                                      │
│  │   │   └── holdings[] (보유 종목)                            │
```

### Step 2: 사용자 인증 Store (authStore) 구현

**목표**: 로그인/로그아웃 상태 관리, 현재 사용자 정보 유지

**생성 파일**: `src/store/authStore.js`

```javascript
import { create } from 'zustand'
import { persist } from 'zustand/middleware'

export const useAuthStore = create(
  persist(
    (set) => ({
      currentUser: null,     // { id, name, email }
      isLoggedIn: false,

      login: (user) => set({
        currentUser: { id: user.id, name: user.name, email: user.email },
        isLoggedIn: true,
      }),

      logout: () => set({
        currentUser: null,
        isLoggedIn: false,
      }),
    }),
    { name: 'auth-storage' }
  )
)
```

**주의사항**:
- MVP 단계에서는 LocalStorage 기반 간단한 인증 (서버 없음)
- 비밀번호 검증은 샘플 데이터와 대조 (평문 비교)

### Step 5: 유틸리티 함수 보강

**목표**: 계좌 단위 / 전체 합산 비중 계산, 섹터별 그룹핑, 일간 변동 계산

**수정 파일**: `src/utils/calculator.js`

```javascript
// 기존 함수에 추가:

// 여러 계좌의 종목을 합산하여 비중 계산
export const calcAllocation = (holdings, exchangeRate = 1350) => {
  // holdings: 단일 계좌 또는 여러 계좌 합산 배열
  const totalKRW = holdings.reduce((sum, h) => {
    const value = h.quantity * h.currentPrice
    return sum + (h.currency === 'USD' ? value * exchangeRate : value)
  }, 0)

  return holdings.map(h => {
    const valueKRW = h.currency === 'USD'
      ? h.quantity * h.currentPrice * exchangeRate
      : h.quantity * h.currentPrice
    return {
      ticker: h.ticker,
      name: h.name,
      value: valueKRW,
      weight: totalKRW > 0 ? (valueKRW / totalKRW) * 100 : 0,
      sector: h.sector,
      market: h.market,
    }
  })
}

// 계좌별 비중 계산 (계좌 간 자산 비중)
export const calcAccountAllocation = (accounts, exchangeRate = 1350) => {
  return accounts.map(acc => {
    const holdingsValue = acc.holdings.reduce((sum, h) => {
      const value = h.quantity * h.currentPrice
      return sum + (h.currency === 'USD' ? value * exchangeRate : value)
    }, 0)
    const cashValue = (acc.cashKRW || 0) + (acc.cashUSD || 0) * exchangeRate
    return {
      accountId: acc.id,
      accountName: acc.accountName,
      accountType: acc.accountType,
      totalValue: holdingsValue + cashValue,
    }
  })
}
```
