## STEP 1 — journalStore 구현

> **파일**: `src/store/journalStore.js` (신규)

### 파일 생성 확인

- ✅ `src/store/journalStore.js` 파일 생성됨

### 심리 카테고리 상수

- ✅ `BUY_PSYCHOLOGY` 배열 정의 (7개 항목)
- ✅ `SELL_PSYCHOLOGY` 배열 정의 (6개 항목)

### 상태 & 미들웨어

- ✅ `zustand` + `persist` + `immer` 미들웨어 적용
- ✅ `entries: []` 초기 상태 정의
- ✅ LocalStorage key: `journal-storage` 로 persist 설정

### 액션

- ✅ `addEntry(entry)` — `crypto.randomUUID()`로 id 자동 생성
- ✅ `updateEntry(id, updates)` — id로 항목 찾아 업데이트
- ✅ `deleteEntry(id)` — id로 항목 제거

### 셀렉터

- ✅ `getEntriesByTicker(ticker)` — 특정 종목 일지만 반환
- ✅ `getEntriesByPsychology(psychology)` — 특정 심리 유형만 반환
- ✅ `getEntriesByDateRange(start, end)` — 기간 내 일지 반환
- ✅ `getProfitByPsychology()` — 심리 유형별 건수, 합계 손익, 평균 손익 반환
- ✅ `getSummaryStats()` — 전체 통계 (총 거래 건수, 매수/매도 건수, 총 손익)

### 동작 검증 (브라우저 콘솔)

```javascript
// 개발자 도구 콘솔에서 확인
const store = window.__journalStore__ // 또는 React DevTools
```

- ✅ `addEntry()` 호출 후 `entries` 배열에 항목 추가됨
- ✅ 새로고침 후 `entries` 데이터 유지됨 (LocalStorage persist 동작)
- ✅ `getProfitByPsychology()` 데이터 없을 때 빈 배열 반환 (에러 없음)
